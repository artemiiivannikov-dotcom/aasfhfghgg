# Weather Diary

Автор: Иванников Артемий

## Описание
Приложение для ведения личного дневника погоды. Позволяет фиксировать температуру, дату и погодные условия, а также фильтровать историю записей.

## Функции
- Добавление записей с валидацией данных.
- Сохранение истории в JSON файл.
- Фильтрация по минимальной температуре.

## Запуск
1. Установите Python 3.x.
2. Запустите файл командой:
   `python weather_diary.py`

## Тестирование
1. Попробуйте ввести текст вместо температуры — программа покажет ошибку.
2. Введите "10.10.2023", "15", "Солнечно" и нажмите добавить.
3. Введите в фильтр "20" — запись должна скрыться.
